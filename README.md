My personal dotfiles
--------------------

This repo is my hope that I could make replicatable dotfiles and a more managable
system. The system I'm running (as of 25.9.2021. is a complete mess IMO). By
sharing dotfiles I think I will subconciously think of a better setup.

The issues are here so you can easily mock me. Enjoy.

Installation
------------

In case you're actually interested in installing these, just assume the each of
the directories in this repo has a dot in front of it. Then just link them using
``ln`` or any similar means to your actual system's ``$HOME`` directory.

I'll probably make a script that deals with the setup too if I'm not too lazy.
